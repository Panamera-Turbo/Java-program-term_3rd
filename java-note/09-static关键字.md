# static关键字
-   修饰类的静态成员

## 静态变量

-   共享的变量用`static`修饰，即静态变量.

-   同一个类的不同实例对象若共用同一个静态变量，则一个类将其改变，另一个类静态变量也随之改变

-   当类首次被加载时，静态变量就被分配到内存中，知道程序结束才释放

## 静态常量

-   用`final static`修饰
-   可以被多个类共享

## 静态方法

-   无法或不想创建类的对象时，使用静态方法
-   调用类的静态方法，无需创建类的对象

## 静态代码块

-   用`static{''代码''}`修饰
-   在声明时就会开始运行
-   只运行一次
-   非静态代码块会在每次new对象的时候先于构造方法运行
