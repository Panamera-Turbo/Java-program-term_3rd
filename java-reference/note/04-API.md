# API
## 使用步骤
1. 看帮助文档
2. 找索引，看到输入框
3. 输入查找内容
4. 看包。除了`java.lang`都要导包
5. 学习使用并应用

## 常用API
- 引用类型的一般使用步骤
    1. 导包
       - 如果使用的目标类和当前类位于一个包下，可以不写
       - java.lang不需要导包语句
        
    2. 创建
    3. 使用

### Scanner-输入
- 从键盘输入：`System.in`
    - `nextInt()`:获取输入的int（10进制）
    - `next()`:获取输入字符串


### Random类
- 创建伪随机数
- `nextInt()`:获取输入的int（10进制）
- `nextInt(int a)`:获取[0, a-1]的整数
